# KikuAI Brand Guidelines

## Brand Mark v1 (Folded K)

### Safe Area
Minimum clear space around the mark equals 1/6 of the mark's width.

### Minimum Sizes
- Favicon: 14px
- UI: 24px
- OG images: 96px

### Color Usage
- **Background:** #111315 (not pure black)
- **Text:** #EDEDED (primary), #B6B8BE (secondary)
- **Accent:** #9A7DF9 (use sparingly for links/focus/CTAs)

### Prohibited
- ❌ Gradients
- ❌ Shadows
- ❌ Outlines/strokes
- ❌ Negative on noisy backgrounds

### Files Included
- `kikuai-k-mark.svg` - White mark (for dark backgrounds)
- `kikuai-k-mark-dark.svg` - Dark mark (for light backgrounds)
- `kikuai-wordmark.svg` - "KikuAI" wordmark with small caps "AI"
- `favicon.svg` - SVG favicon
- `favicon-16x16.png`, `favicon-32x32.png` - PNG favicons
- `apple-touch-icon.png` - 180x180 iOS icon
- `maskable-icon.png` - 512x512 maskable icon
