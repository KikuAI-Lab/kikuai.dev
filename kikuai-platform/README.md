# kikuai-platform

Docker Compose stacks for reverse proxy, observability, databases, and templates.

Stacks (planned):
- reverse-proxy: Caddy (Cloudflare DNS-01, labels-based routing)
- observability: Uptime Kuma, Prometheus, cAdvisor, Node Exporter, Loki, Promtail, Grafana
- data: PostgreSQL 16, Redis
- optional: Watchtower

Usage: see compose.yml (to be added) and labels examples.
